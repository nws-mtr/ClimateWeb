Notes

This page summarizes daily climate observations per station, including temperature extremes and precipitation progress for the current water year.

Percent of normal compares the current water year precipitation to the long-term normal. Values below 75% are flagged as below normal, 75-125% as near normal, and above 125% as above normal.

A value of 9999 in the source data indicates missing information; this dashboard displays those as an em dash (—).
